# 웹 소켓을 이용한 채팅 애플리케이션

로그인을 하고 인증된 사용자만 채팅방에 입장하여 채팅이 가능한 서비스입니다.

채팅 서비스를 만들어보고 싶었고, 제가 원하는 기능을 추가해보고자 개인으로 제작하게 되었습니다.
